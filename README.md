# 北交大课程平台青春版

[![Vue3](https://img.shields.io/badge/Vue3-4FC08D?logo=vuedotjs&logoColor=white)](https://vuejs.org/)
[![TypeScript](https://img.shields.io/badge/TypeScript-3178C6?logo=typescript&logoColor=white)](https://www.typescriptlang.org/)

一个基于官方接口重构的课程平台，适配移动设备和提供更符合学生需求的功能放置。
功能概览：
自动登录、课件下载、课件快速预览、成绩排名、优秀作业参考、增强作业查看、作业文件预览、文件提交、课表查询等常用功能的体验。

**国内访问:**（建议ipv6访问）
[course.bjtu.top](https://course.bjtu.top/)

>感谢好朋友 上条当咩 提供的子域名和服务器。

## 🌟核心功能介绍


#### 自动登入和会话保持
我不想输密码，于是有了这个功能：每次打开网页自动尝试重连或者重新登入对话。
#### 每日课表
1.某个假期知道要调课，但不知道那天补星期几的课🥲。
2.有门课一周两节，但上课教室不一样😢。
于是新增 查看当日剩余课程、时间倒计时

#### 待办列表
有的老师布置作业都不吱声，导致我几次差点在全然不知的状态下错过截止时间。于是有了
- 方便的查看和提交待完成的任务
- 截止倒计时(感受到紧迫感了吗)
- 同时保留了通过课程分类查看作业

#### 提交作业功能增强
原先交作业提交多个附件时，得一个个点（没坐，甚至不能拖拽上传），于是有了：
- 继承作业查看，预览和提交功能，增强：新增拖动上传，多文件一起上传

- 新增删除已提交作业功能（我一人机兄弟交错作业后发现不能重复提交，求着我加了这个功能）

- 覆盖提交功能（强制覆盖原作业，类似先删除再提交。但如果截止时间过了，单纯的提交功能将被禁止，覆盖功能依旧开放）

- 过期补交功能。本来可以提交接口就能直接补交
> 作业过期后止提交的实现是--前端不显示提交按钮！但[@上条当咩](https://nimisora.icu/)做了浏览器插件显示补交按钮后，平台转眼就在后端加上了时间校验。
> 但之前提到，后端不鉴权。于是聪明的我想到另一个思路：利用神秘api登录老师的账号，抓取老师端的补交接口。果不其然，学生账号同样顺利调用。

####  查看已提交的作业
作业太难做不出来，我需要拜读一下同学的高质量作业，因此悄咪咪加了功能（别打我🤕，自从我同班同学知道了之后，再也没有人提前交作业了）
#### 基础课件功能增强
 原平台每次只能预览一个课件，切换课件要1返回、2重新加载全部课件列表、3再点开新课件；而且移动端适配简直是依托💩。于是有了：
- 附带查看、下载未公开和不允许下载的课件😝
- 提供pdfjs高版本特性，如大纲、高亮画笔、文字搜索、全屏放映等功能
